# キャットフード評価iOSアプリ 企画書

## 1. 企画概要

### アプリ名（仮）
**CatFood Scanner（キャットフードスキャナー）**

### コンセプト
キャットフードのパッケージや成分表を撮影するだけで、AIが自動解析して点数評価とGood/Badポイントをフィードバック。専門知識がなくても、愛猫に最適なフード選びができる「AI栄養士」アプリ。

### ターゲットユーザー
- キャットフード選びに悩んでいる飼い主
- 成分表を見ても判断できない初心者〜中級者
- 愛猫の健康を気にかけているが忙しい人
- 年齢層：20代〜50代、特に30〜40代

### 主な機能
1. **写真撮影・アップロード** - パッケージや成分表を撮影
2. **AI自動解析** - OCRで成分抽出 + AIによる栄養評価
3. **スコア表示** - 総合点（100点満点）+ カテゴリ別評価
4. **Good/Badコメント** - 具体的な改善ポイントと優れた点
5. **履歴管理** - 過去に解析したフードの記録
6. **マイキャット登録** - 年齢・体重に応じた評価カスタマイズ

## 2. 機能要件

### 必須機能（MVP）
| 機能 | 詳細 |
|------|------|
| 写真撮影/アップロード | カメラ撮影 or ライブラリから選択 |
| AI画像解析 | OCRで成分表テキスト抽出 |
| 栄養評価AI | 成分から自動スコアリング |
| スコア表示 | 総合点（100点満点）、栄養バランス・原材料品質・添加物の3軸評価 |
| Good/Badコメント | AIが生成する具体的なフィードバック（3〜5項目ずつ） |
| 解析履歴 | 過去の解析結果を一覧表示、お気に入り登録 |
| マイキャット登録 | 名前、年齢、体重、健康状態（1匹）→ パーソナライズ評価 |

### 評価項目の詳細
```
【総合スコア 100点満点】
├─ 栄養バランス (40点): タンパク質・脂質・炭水化物の比率
├─ 原材料品質 (40点): 主原料の質、副産物の有無
└─ 添加物評価 (20点): 着色料・保存料・香料の種類

【Good/Badコメント例】
✅ Good
- 高品質なチキン生肉が主原料
- グレインフリーで消化に優しい
- オメガ3脂肪酸が豊富

❌ Bad
- 穀物の割合が高く猫に不向き
- 人工着色料（赤色102号）使用
- タンパク質含有率がやや低い
```

### 今後の拡張機能
- **代替商品レコメンド**: より良いフードの提案（アフィリエイトリンク付き）
- **フード比較機能**: 最大3商品を並べて比較

## 4. KPI・成功指標

### 初期目標（リリース後3ヶ月）
- ダウンロード数: 2,000DL
- 月間アクティブユーザー: 500名
- 解析実行回数: 3,000回
- 有料プラン転換率: 5%（月額480円想定）
- NPS（推奨度）: 40以上

### 収益化モデル
- **無料プラン**: 月5回まで解析無料
- **プレミアムプラン（月額480円 or 年額4,800円）**:
  - 解析回数無制限
  - 複数猫登録（最大5匹）
  - 詳細PDFレポート出力
  - 広告非表示
- **将来的な収益源**:
  - 代替商品レコメンドのアフィリエイト収益（Amazon/楽天）
  - フード比較機能での優良商品紹介

## 5. リスク・課題

### 技術リスク
- **OCR精度**: 手書きや不鮮明な写真で誤認識 → 再撮影ガイド、手動入力オプション
- **AI評価の信頼性**: 獣医学的根拠の不足 → 免責事項明記、獣医師監修の検討
- **API費用**: AI APIは高額になる可能性 → キャッシュ活用、コスト最適化

### ビジネスリスク
- **ユーザー獲得**: ニッチ市場 → SNS（特にInstagram）でのインフルエンサーマーケティング
- **法的問題**: 医療アドバイスと誤認 → 「参考情報であり診断ではない」旨を明記
- **競合**: 類似アプリ出現 → AI精度とUXで差別化

### 対策
- MVP段階では獣医師1名に監修依頼、信頼性向上
- チュートリアルで「あくまで参考情報」と強調
- 初期はインフルエンサー50名に無料提供しレビュー獲得
